All you need to play this game is a python interpreter (recomanded: Pycharm or Visual Studio Code)

Step by Step guide: 
1. Install Files:
   Install and import all the game files to you're project (main.py, score.py , ect)
2. Install Turtle Graphics:
   Turtle Graphics should be preinstalled with python, if you don't have it installed, run the folowing command in you're terminal: pip install turtle or pip install PythonTurtle
3. Play the game:
   Invite a friend over and play the game! W and S to control left paddel | ARROW UP and ARROW DOWN to control right paddel
4. Change code:
   Feel free to tweak the code to you're liking! Here is the official turtule graphics documentation: https://docs.python.org/3/library/turtle.html
